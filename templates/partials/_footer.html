User Roles in PersonFinder

Roles define who can search, register, and administer the system.

Super Admin

Highest access

Owns security, environment configuration, and global policies.

  • • Configure infrastructure & environment variables
  • • Manage admin roles & access controls
  • • Oversee critical data protection decisions

Admin

Operational

Manages people records, cases, and user activity.

  • • Review and manage registrations
  • • Monitor search logs & feedback
  • • Manage user accounts & access issues

User

Community

Frontline role for registering and searching missing persons.

  • • Register missing persons with details and photos
  • • Search the PersonFinder database
  • • Provide case updates & additional information

Role changes are intentionally limited to preserve privacy, auditability, and trust.
